MAC Address Filter
Allow or disallow client connections by configuring MAC address filtering. For example: 40:4D:8E:6D:80:7D.
(1)Disable: Disable MAC address filtering.
(2)Deny: If the client's MAC address exists in the MAC address list, deny the client to use the device.
MAC filtering
IP Address Filter
Allow or disallow client connections by configuring IP address filtering. For example: 192.168.2.35.
(1)Disable: Disable IP address filtering.
(2)Deny: If the client's IP address exists in the IP address list, deny the client to use the device.
IP filtering
URL Address Filter
Allow or disallow client connections by configuring URL address filtering. For example: www.example.com.
(1)Disable: Disable URL address filtering.
(2)Deny: If the client's URL address exists in the URL address list, deny the client to use the device.
URL filtering

Recovery
After recovery, the configuration of the router will be restored to the factory state, so use it with caution! It is recommended that you back up your current configuration before a factory restore.
Backup Configuration
Export and save the current configuration of the router in your computer. If you need to restore this configuration in the future, you can simply import the backup file. Tip: The exported profile encrypts your personal data, so please keep it safe.
VPN
A virtual private network (VPN) is a private network established based on an existing public network. A VPN uses security technigues such as encryption and identity authentication to ensure confidentiality and security of communication data. For example, to access the company network from your computer, you can connect your computer to the company's VPN server. By doing this, you can easily access secure network resources on the company network
PORT
Port Forwarding refers to the access of the external network to the specified port service of the router, which is forwarded to the specific port of a networking device in the internal LAN through the router (the set port number can be the same or different), so as to realize the normal access of the router's external network to the internal network service.
